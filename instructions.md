# Подсказка оп GIT


Создание репозитория:

``` sh
git init
```

Добавление файла:

``` sh
git add
```

Коммитить файл в репозиторий:

``` sh
git commit -m "Сообщение об изменении"
```

Показать журнал изменений:

``` sh
git log
```

Показать журнал изменений в одной линии:

``` sh
git log --oneline
```

Изменение бранча (HEAD):

``` sh
git checkout
```

Проверка статуса:

``` sh
git status
```
